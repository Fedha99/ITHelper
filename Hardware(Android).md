#komponen 
## Secara Garis Besar Ada 3 Komponen
### 1. Hardware Internal (Jeroan)

- **SoC (System on Chip):** Otak utama yang isinya CPU, GPU, dan modem jadi satu.
    
- **[[Random Access Memory - RAM]]:** Tempat penyimpanan data sementara aplikasi yang sedang berjalan.
    
- **Internal Storage (UFS/eMMC):** Memori permanen tempat nyimpen OS, foto, dan aplikasi.
    
- **Baterai (Lithium-Ion/Polymer):** Sumber daya utama perangkat.
    
- **[[Motherboard (Mainboard)]]:** Papan sirkuit tempat semua komponen terhubung.
    

### 2. Komponen Input & Output (Interface)

- **Layar (LCD/AMOLED):** Panel visual sekaligus sensor sentuh (digitizer).
    
- **Modul Kamera:** Sensor gambar dan lensa (depan & belakang).
    
- **Speaker & Microphone:** Komponen untuk suara dan input audio.
    
- **Port USB (Type-C/Micro):** Untuk pengisian daya dan transfer data.
    
- **Vibration Motor:** Komponen penghasil getaran (haptic).
    

### 3. Konektivitas & Sensor

- **Antena (WiFi, Bluetooth, 4G/5G):** Untuk komunikasi nirkabel.
    
- **Sensor Biometrik:** Fingerprint scanner atau Face ID.
    
- **Sensor Lingkungan:** Akselerometer, Giroskop, Proximity, dan Kompas.
